# About

We publish **pregame** estimates of MLB game totals and winners — built for people who want to see how an ensemble model reasons about a slate, not for selling picks.

The product combines open baseball data (game logs, schedules, weather, and sidecars from public APIs) with **ensemble machine learning**: thirteen independent members, each capturing a different signal, blended with weights tuned on a validation season and evaluated on a held-out test year. We report those holdout metrics openly and explain our methodology so you can judge the limits of the model yourself.

Built with open data and ensemble ML — no hype, no guarantee of future edge.
